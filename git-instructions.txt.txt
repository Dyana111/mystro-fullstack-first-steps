instructions:


            new project                     --->    git init 
            track changes                   --->    git add file_name
            save backup                     --->    git commit -m "massage like what have
                we done in this version"
            see history                     --->    git log   "to see all versions"
            see any changes                 --->    git status
            upload changes to github        --->    git push  "upload to cloud"
            get changes from github         --->    git pull
            to move between branches        --->    git checkout (if we put 
                                                        "-b branchName" git will create a
                                                             branch under this name in case there was no)
